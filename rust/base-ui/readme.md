Очистка
--------------

    cargo clean
    
    

Запуск сервера
------------------
    
    trunk serve




Сборка
----------------

    trunk build



Полезные опции команды trunk build


| Опция | Описание |
|---|---|
| --release | Сборка с оптимизациями (аналог cargo build --release) |
| --dist <path> | Указать другую папку для вывода (по умолчанию dist) |
| --public-url <url> | Указать базовый URL для ассетов (полезно для деплоя | на подпапку)
| --no-sri | Отключить Subresource Integrity (хэши файлов) |
| --watch | Следить за изменениями и пересобирать автоматически |
